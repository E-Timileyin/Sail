### Prerequisites

- Node.js 18+ and Yarn

### Installation

1. **Clone the repository**
   ```sh
   git clone <repository-url>
   cd sail-website
   ```

2. **Install dependencies**
   ```sh
   yarn install
   ```

3. **Start the development server**
   ```sh
   yarn dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.

4. **Build for production**
   ```sh
   yarn build
   yarn start
   ```
